# Challenge 1: Sum Two Numbers

## Objective

Write a program that reads two integers from the user and prints their sum.

## Requirements

1. Use `std::cin` to read two integers
2. Calculate their sum
3. Use `std::cout` to display the result

## Example

**Input:**
```
5
3
```

**Output:**
```
8
```

## Hint

Remember to declare variables to store the numbers before reading them!
