# Challenge 4: Temperature Converter

## Objective

Create a program that converts temperature from Celsius to Fahrenheit.

## Requirements

1. Ask user for temperature in Celsius
2. Convert to Fahrenheit using the formula: F = (C × 9/5) + 32
3. Display the result

## Formula

**Fahrenheit = (Celsius × 9/5) + 32**

## Examples

- 0°C = 32°F (freezing point of water)
- 100°C = 212°F (boiling point of water)
- 37°C = 98.6°F (normal body temperature)

## Sample Input/Output

```
Enter temperature in Celsius: 25
25°C = 77°F
```

## Bonus Challenges

1. Also convert Fahrenheit to Celsius
2. Add Kelvin conversions
3. Validate input (no temperatures below absolute zero)

## Hint

Remember to use `double` for decimal temperatures!
