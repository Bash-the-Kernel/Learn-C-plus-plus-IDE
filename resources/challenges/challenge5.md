# Challenge 5: Grade Calculator

## Objective

Calculate the average of 5 test scores and assign a letter grade.

## Requirements

1. Read 5 test scores from the user
2. Calculate the average
3. Assign letter grade based on average
4. Display both average and letter grade

## Grading Scale

- **A**: 90-100
- **B**: 80-89
- **C**: 70-79
- **D**: 60-69
- **F**: Below 60

## Sample Input/Output

```
Enter 5 test scores:
Score 1: 85
Score 2: 92
Score 3: 78
Score 4: 95
Score 5: 88

Average: 87.6
Letter Grade: B
```

## Algorithm

1. Declare variables for 5 scores and total
2. Use loop or individual inputs to read scores
3. Calculate average = total / 5
4. Use if-else statements for letter grade

## Bonus Features

- Validate scores are between 0-100
- Show which scores are above/below average
- Calculate GPA points (A=4, B=3, C=2, D=1, F=0)
