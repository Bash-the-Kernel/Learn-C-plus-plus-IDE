# Lesson 1: Hello World

## Introduction

Every C++ program begins with a `main()` function. This is where your program starts executing.

## The Basic Structure

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

## Breaking It Down

- `#include <iostream>` - Includes the input/output stream library
- `int main()` - The main function that returns an integer
- `std::cout` - Standard character output (console output)
- `<<` - Stream insertion operator
- `std::endl` - End line (newline character)
- `return 0` - Indicates successful program execution
